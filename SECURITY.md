# Security Policy

## Official Repositories

The only official repositories for this project are:

- **Ocelot Wallet**: https://github.com/ryo-zen/ocelot-wallet
- **ZeiCoin**: https://github.com/ryo-zen/zeicoin

Any other repository claiming to be Ocelot Wallet or ZeiCoin is not affiliated with this project. Always download from the official repositories listed above.

## Verifying Releases

All official releases are published exclusively at:
https://github.com/ryo-zen/ocelot-wallet/releases

Releases contain source code only. Build from source for maximum assurance.

## Ownership

The ZeiCoin project is operated by ryo-zen. Official domains:
- https://zei.network
- https://zeicoin.com

## Reporting a Vulnerability

To report a security vulnerability in Ocelot Wallet, please open a private security advisory at:
https://github.com/ryo-zen/ocelot-wallet/security/advisories/new

Do not report security vulnerabilities via public GitHub issues.
